一、vi和vim的三种常见模式
    1.正常模式
           一开始进入的就是正常模式，可以移动光标，可以删除内容，复制粘贴等。
    2.插入模式/编辑模式
           输入i、I、o、O、a、A、r、R等任意一个字符都可进入编辑模式，一般来说按i即可。
    3.命令行模式
          在该模式下可以执行相关指令，完成读取、存盘、替换、离开vim、显示行号等活动。
    提示：在编辑模式下，按esc进入正常模式和命令行模式。

    快捷键：
        数字yy	表示复制当前行向下的x行。
        p                      表示粘贴。
        数字dd             表示删除当前行向下的x行。
        /关键字            表示查找该关键字，输入n即查询下一个。
        ：set nu           表示设置行号，nonu表示不设置行号。
        G                      移动到末行
        gg                    移动到首行
        u                      在正常模式下表示撤销
        数字 Shift+g    在正常模式下，表示移动至第i行

二、关机重启
    1.shutdown
        shutdown -h now	表示立即关机
        shutdown -h 数字	表示i分钟后关机
        shutdown -r now             表示立即重启

    2.halt
        直接使用，表示立即关机

    3.reboot
        直接使用，表示立即重启

    4.sync
        直接使用，表示将未保存的数据保存。在关机或重启时最好先执行一次。

三、用户的登陆与注销
    1.su 用户名	表示切换用户，用户权限高切换为低可以不输入密码，反之不行。

    2.logout	表示注销用户，在图形用户界面下输入该命令无效。

四、用户管理
    1.useradd [选项] 用户名	添加用户
        -d 主目录路径	指定目录
        -g 组名	指定组
    如果没有指定组，则自动产生一个与用户名相同的组，并加入其中。没有指定目录，则自动在home下生成一个与用户名相同的目录。
    
    2.passwd 用户名		修改和设置用户密码

    3.userdel 用户名		删除用户
        如果只是简单的 userdel 用户名 只会删除用户，不会删除用户的目录，如果要删除用户及用户主目录需要输入 userdel -r 用户名

    4.id 用户名		查询用户
        如果用户不存在会显示无此用户，如果用户存在会显示用户id以及用户所在组的id

    5.groupadd 组名		添加组

    6.groupdel 组名		删除组

    7.usermod -g 组名 用户	修改用户的所在组

五、实用指令
    1.运行级别
        linux的运行级别有7种：
            0：关机
            1：单用户
            2：多用户无网络
            3：多用户有网络（命令行）
            4：系统未使用保留给用户
            5：图形界面
            6：系统重启
        可以通过 init 级别 来切换，也可以通过修改/etc/inittab/的id：级别：initdefault来修改其默认级别。

    面试题：如果忘记了root用户的密码如何找回？
        首先重启系统，在重启时点击f2或f10进入系统，然后通过切换运行等级进入单用户模式，进入后就是root用户，然后通过passwd修改root用户的密码。
        如果遇到/etc/inittab文件的运行级别为0、6或超出范围的数，也可以通过单用户模式，进入命令行对inittab文件进行修改。

    2.man [命令或配置文件]	获取帮助信息

    3.help 命令		获取shell内置命令的帮助信息

    4.通过百度也可以查询相关指令的信息

    5.pwd			显示当前工作目录的绝对路径

    6.ls [选项] [目录或是文件]	显示该目录或文件的信息
        -a	显示当前目录的所有文件和目录，包括隐藏的。
        -l	以列表的方式显示信息
        -ahl	查看文件所有者

    7.cd [参数]		进入到指定目录
        参数可以是相对路径也可以是绝对路径
        cd 或 cd ~	回到用户的主目录
        cd ..		回到上一级目录

    8.mkdir [选项] 要创建的目录	创建目录
        -p	创建多级目录

    9.rmdir [选项] 要删除的目录	删除目录
        rmdir删除的是空目录，如果该目录下有内容则无法删除。

    10.touch 文件名		创建文件

    11.cp [选项] source dest	复制文件
         -r	复制整个文件
         \cp	强制覆盖

    12.rm [选项] 文件或目录	删除文件或目录
         -r	删除整个文件
         -f	强制删除不提示

    13.mv指令
         mv 原文件名 新文件名	重命名
         mv 原文件 新路径	移动文件

    14.cat [选项] 要查看的文件	查看文件内容
         -n	显示行号

    15.more 文件名		分页查看文件内容
          快捷键：
              空格键	下一页
              回车键	下一行
              q	退出
              Ctrl+F	下一页
              Ctrl+B	上一页

    16.less 文件名		分页查看文件内容，适用与查看大型文件

    17.内容 > 文件名		覆盖原本文件的内容

    18.内容 >> 文件名	在原文件的末尾追加内容

    19.echo [选项] [输出内容]	输出内容到控制台

    20.head 文件		查看文件头10行，默认是10行。
        head -n 数字 文件	查看文件头i行。

    21.tail 文件		查看文件末尾10行，默认为10行。
        tail -n 数字 文件	查看文件末尾i行。
        tail -f 文件		实时追踪文档的所有更新。

    22.ln [原文件或目录] [软链接名]	给源文件创建一个软链接，相当于快捷图标。

    23.history		查看历史指令

    24.date 		显示当前日期
        date +%Y		显示当前年份
        date +%m		显示当前月份
        date +%d		显示当前日
        date ”+自定义“	显示当前自定义日期
        date -s 字符串时间	设置日期

    25.cal [选项]		显示当前日历

    26.find [搜素范围] [选项]	搜索文件
        -name	通过名字
        -size		通过文件大小
        -user	通过所有者

    27.locate 搜索文件	通过数据库查询，快，但是第一次执行前要使用updatedb创建locate数据库

    28.grep [选项] 查找内容 源文件	过滤查询
        -n	显示行号
        -i	忽略大小写

    29.gzip/gunzip		压缩文件/解压文件
        这两个指令执行后源文件都不会保留下来

    30.zip/unzip		压缩文件/解压文件
        zip [选项] xxx.zip
            -r	递归压缩
        unzip [选项] xxx.zip
            -d<目录>指定位置

    31.tar [选项] xxx.tar.gz	压缩和解压
        -c	产生.tar打包文件
        -v	显示详细信息
        -f	指定压缩后的文件名
        -z	打包同时压缩
        -x	解包.tar文件
            -zcvf	打包
            -zxvf	解包

    32.chown 用户名 文件名	修改文件的所有者

    33.chgrp 组名 文件名	修改文件的组

    34.usermod -g 组名 用户名	修改用户的所在组
        usermod -d 目录名 用户名	修改用户的主目录

    35.chmod u=[权限]，g=[权限]，o=[权限] 目录或文件	修改目录或文件的权限
        u表示所有者	g表示组	o表示其他组	a表示所有人
        三种权限：
            r	读
            w	写
            x	执行
        顺序rwx，将其转化为数字：r=4，w=2，x=1，所以rwx也可以用数字7表示，即二进制的111
        chmod 数字（三位数） 文件名	修改文件或目录权限
        chmod (u/g/o/a)(+/-)[权限] 文件名		给(所有者/组/其他组/所有人)（加上或减去）某个权限

    36.crontab [选项]		任务调度
        -e	编辑crontab定时任务
            * * * * * 命令。第一个*代表分，第二个*代表时，第三个*代表日，第四个*代表月，第五个*代表周几
        -l 	查询crontab任务
        -r	删除当前用户的所有crontab任务
